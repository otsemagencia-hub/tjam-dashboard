# 📊 Dashboard TJ AM — Estudos

Dashboard visual de acompanhamento de estudos para o concurso do **Tribunal de Justiça do Amazonas (TJ AM)**.

**Banca:** CEBRASPE (Cespe/UnB) · **Cargo:** Assistente Judiciário · **Vagas:** 400

## Funcionalidades

- **Visão geral** — questões resolvidas, taxa de acerto, matérias, recursos
- **Cobertura do Edital** — progresso por matéria (Peso 1 e Peso 2)
- **Assuntos pendentes** — o que ainda falta estudar em cada matéria
- **Revisão Espaçada** — tópicos que precisam de revisão urgente
- **Linha do Tempo** — sessões de estudo ao longo do tempo

## Tecnologias

- HTML / CSS / JS puro (zero dependências)
- Design responsivo · Tema escuro amazônico
- Google Fonts: Plus Jakarta Sans + Instrument Serif

## Acessar

Abra o arquivo `index.html` em qualquer navegador moderno.
